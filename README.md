# 📚 HQ Livre

Aplicativo mobile desenvolvido em **React Native + Expo** para leitura e gerenciamento de HQs e Mangás.

O projeto possui uma interface simples, moderna e intuitiva, permitindo visualizar obras, pesquisar títulos, favoritar conteúdos e acessar informações detalhadas de cada publicação.

---

## 📱 Telas do aplicativo

### 🏠 Home

Tela inicial do aplicativo.

Funcionalidades:

- Apresentação da plataforma HQ Livre;
- Carrossel de destaques;
- Seção de lançamentos;
- Navegação inferior.

---

### 📖 Catálogo

Tela responsável pela exibição das obras cadastradas.

Funcionalidades:

- Campo de pesquisa;
- Filtros por categoria;
- Listagem das obras;
- Adição de novas obras.

---

### 📄 Detalhes

Tela que apresenta as informações completas da obra.

Informações exibidas:

- Capa;
- Título;
- Autor;
- Gênero;
- Quantidade de capítulos;
- Status;
- Avaliação;
- Sinopse.

Ações disponíveis:

- Editar;
- Excluir;
- Favoritar.

---

### ⭐ Favoritos

Tela responsável por armazenar as obras favoritas do usuário.

Funcionalidades:

- Pesquisa;
- Filtro por categorias;
- Acesso rápido às obras salvas.

## ⚙️ Instalação

### 1. Clone o repositório

```bash
git clone git@github.com:Leroy-jester/HQlivre.git
```

### 2. Entre na pasta do projeto

```bash
cd HQlivre
```

### 3. Instale as dependências

```bash
npm install
```

### 4. Execute o aplicativo

```bash
npx expo start
```

Após iniciar, utilize:

- Expo Go (Android);
- Emulador Android Studio;
- Simulador iOS.
